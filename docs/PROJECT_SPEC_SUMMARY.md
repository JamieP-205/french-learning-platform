# French for Life - Project Summary

Build a polished, practical, adaptive French-learning web app that teaches real communication through verified scenario-led content, structured practice, spaced review, mistake repair, and guarded AI tutoring.

The app teaches the person rather than a fixed course. Daily sessions combine review, useful new language, listening/reading input, speaking or writing output, targeted repair, and an encouraging completion moment. Prioritise functional French, retrieval practice, spaced repetition, register awareness, and visible skill progress over XP or generic chat.

The first vertical slice is one A1 mission: **Introduce yourself and talk about your day**. It includes onboarding, Today, mixed activities, deterministic validation, mistakes, review scheduling, progress, browser speech/listening practice, streak/garden retention, private friend-code co-op, deterministic roleplay, an optional grounded tutor, and healthy motivation. Defer subscriptions, native apps, broad curriculum import, production speech scoring, and open-ended AI curriculum generation.

AI is a component rather than the product brain. Store learner memory in the app database, retrieve exact verified records for factual tasks, validate structured JSON before use, and use a confident safe-standard fallback for unsupported questions.
